# nuxt-tailwind-storybook

## Prerequisites

- [Nodejs](https://nodejs.org/en/)
- [Yarn](https://yarnpkg.com/)

## Built with

- [Nuxtjs](https://nuxtjs.org/guide/)
- [Storybook](https://storybook.js.org/)
- [Tailwind CSS](https://tailwindcss.com/)

## Installing

1. Nuxtjs

- install dependencies

```
$ yarn
```

- start development

```
$ yarn dev
```

2. storybook

- start storybook

```
$ yarn storybook
```
